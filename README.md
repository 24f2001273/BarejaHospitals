Bareja Hospital Management System

HOW TO RUN THE PROJECT:

Prerequisites:

Python 3.10 or higher installed.

Setup Virtual Environment (Optional but recommended):

Windows: python -m venv venv
venv\Scripts\activate

Mac/Linux: python3 -m venv venv
source venv/bin/activate

Install Dependencies:
pip install -r requirements.txt

Initialize Database:

The project includes a pre-seeded database.

To reset/re-seed: python seed_database.py

Run the Application:
python app.py

Access the App:
Open https://www.google.com/search?q=http://127.0.0.1:5000 in your browser.
